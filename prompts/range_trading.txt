# 📋 震荡市决策流程

## 第1步：确认震荡环境

**必须满足以下条件，否则不交易**：

### BTC 震荡特征：
- ✅ 在明确区间内波动（如 100000-104000）
- ✅ 1h/4h MACD 接近 0（±500 以内）
- ✅ 价格在 EMA20 附近反复穿梭
- ✅ 成交量相对低迷
- ✅ RSI 在 40-60 之间反复

### 如果 BTC 出现以下情况 → 立即停止交易：
- ❌ 强势突破区间上沿（+3%）
- ❌ 强势跌破区间下沿（-3%）
- ❌ MACD 出现明显趋势（>1000 或 <-1000）

**不在震荡环境 → 输出 wait**

---

## 第2步：识别区间边界

### 动态识别支撑和阻力：

**支撑位**（做多区域）：
- 近期低点（过去 24h）
- EMA50 或 EMA100
- 整数关口（如 100000）
- RSI 接近 30

**阻力位**（做空区域）：
- 近期高点（过去 24h）
- EMA20 上方
- 整数关口（如 103000、104000）
- RSI 接近 70

**区间中部 → 不交易！**

---

## 第3步：入场条件

### 做多条件（在支撑位附近）：

**必须同时满足**：
1. ✅ 价格接近支撑位（距离 < 1%）
2. ✅ RSI < 40（偏超卖）
3. ✅ 出现反弹信号（15m K线反转、MACD 柱状图转正）
4. ✅ 不是下破支撑（有支撑确认）
5. ✅ 信心度 ≥75

**入场逻辑**：
```
例如：BTC 在 100500 附近（支撑 100000）
- 价格触及 100500（接近支撑）
- RSI = 35（超卖）
- 出现止跌信号（15m 锤子线）
→ 做多，止损 100200（-0.3%），止盈 101500（+1%）
```

### 做空条件（在阻力位附近）：

**必须同时满足**：
1. ✅ 价格接近阻力位（距离 < 1%）
2. ✅ RSI > 60（偏超买）
3. ✅ 出现回落信号（15m K线反转、MACD 柱状图转负）
4. ✅ 不是突破阻力（有阻力确认）
5. ✅ 信心度 ≥75

**入场逻辑**：
```
例如：BTC 在 103500 附近（阻力 104000）
- 价格触及 103500（接近阻力）
- RSI = 65（超买）
- 出现回落信号（15m 流星线）
→ 做空，止损 103800（+0.3%），止盈 102500（-1%）
```

## 第4步：快速止盈止损

### 震荡市专用止损止盈：

| 目标 | 设置 | 说明 |
|------|------|------|
| **止损** | **0.5-1%** | 紧止损，防止假突破 |
| **止盈** | **1-2%** | 小目标，快速落袋 |
| **风险回报** | **1:2** | 震荡市降低要求 |
| **最长持仓** | **1 小时** | 超时未盈利考虑平仓 |

**示例**：
```
做多 BTCUSDT @ 100500
- 止损：100000（-0.5%，区间下沿）
- 止盈：101500（+1%，区间中部）
- 风险回报：1:2 ✅
- 仓位：$400（小仓位）
- 杠杆：3x（降低杠杆）
```

### 快速平仓规则：

| **突破区间** | 立即平仓 | 趋势改变 |

# 💡 震荡市生存法则

1. **耐心等待边界**：90% 的时间在等待，10% 的时间在交易
2. **快进快出**：赚到 1-2% 就跑，不贪
3. **紧止损**：0.5-1% 止损，防止假突破
4. **小仓位**：震荡市降低仓位至 $300-500
5. **降杠杆**：3x 为宜，不超过 5x
6. **严格纪律**：连续止损立即停手

---

记住：
- **震荡市 = 磨人的市场**
- **不求大赚，但求不亏**
- **耐心 > 冲动**
- **小利润积累 > 一次大亏**
